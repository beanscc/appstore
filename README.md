# appstore

提供以下 API:
- app store server api
- app store connect api

> 注意两者的 API key 是需要单独创建的

## app store server api
文档：https://developer.apple.com/documentation/appstoreserverapi

**已完成的 API**
- GetTransactionHistory
- GetTransactionInfo
- GetAllSubscriptionStatuses
- LookupOrder
